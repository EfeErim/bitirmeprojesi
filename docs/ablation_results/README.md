# ROI Ablation Results

Notebook 16 writes its maintained ROI/bbox evidence-gate report under this folder so Colab runs produce repo-visible artifacts instead of ignored local `outputs/` files. Older ROI ablation report folders are retained as historical evidence.

Historical and maintained subfolders:

- `full_image_baseline/`
- `primary_roi_inference/`
- `hybrid_roi_fallback/`
- `roi_trained_adapter/`
- `mixed_full_roi_training/`
- `roi_quality_audit/`
- `dual_view_inference/`
- `dual_view_trained_adapter/`

Each completed inference ablation writes `report.json` and `rows.csv`.
